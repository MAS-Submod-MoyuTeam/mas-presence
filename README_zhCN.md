## 🌟 功能

* 简单、简约而又极具扩展性的配置
  * 创建无限的配置并在它们之间切换，支持表达式~
  * 使用熟悉的Ren'Py语法对文本中的任何可访问变量进行插值
  * 你可以在文本和条件表达式中使用几个额外的变量
* 开箱即用的配置；只需安装它，就能看到它的状态~
  * 带有大量预制的存在配置，用于许多可能的特殊情况。
    * 日历事件检测和倒计时
    * 专为周年纪念日而设的存在配置
    * 基于一天中的时间的配置~
    * 你和莫妮卡的生日的特殊布局~
  * 包含可在文本行中使用的自定义变量扩展

## ❓ 安装

1. 前往 [最新发布][6] 下滑到资源文件.
2. 下载 `discord-presence-submod-VERSION.zip` 文件.
3. 拖入你的ddlc/mas文件夹

   **NOTE:** *不要拖到`game`里面！*  
4. 你已经准备好了!~

## 📚 自定义

Discord Presence Submod是可以无线定制的，条件驱动的的配置，可插值的文本行和扩展，可以改变选择存在配置的整个方式。

想知道更多？查看 [自定义指南][1]!
